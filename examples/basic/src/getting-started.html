Getting Started with DomPile

Start building static HTML pages the peaceful, minimalist way.

Install

Use it via npx or install globally:

npm install -g dompile

Quick Build

Create your project structure:

/src
  index.html
  header.html
  footer.html

Then:

$ dompile build --input src --output dist

Outputs in dist/, includes resolved recursively, head injections applied.

Preview Mode

Start local server with live rebuild & reload:

$ dompile serve --input src --output dist

Options

  • --dry-run show what would change without writing files
  • --verbose detailed output of includes and processing
  • --includes-dir specify base directory for include files

Why DomPile?

Inspired by early web tools like Dogpile—which gathered search results from many engines into one clean interface—DomPile gathers HTML bits into a unified static page, leaving control to the developer, free from framework complexity.
